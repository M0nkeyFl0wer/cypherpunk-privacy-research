# RAILGUN

## Description
RAILGUN is an on-chain zero-knowledge privacy ecosystem for DeFi, enabling private transactions, shielded balances, and anonymous smart contract interactions directly on Ethereum and EVM-compatible chains using zk-SNARKs.

## Technology
- **Privacy Mechanism**: zk-SNARKs (Groth16)
- **Hash Function**: Poseidon
- **Architecture**: UTXO-based shielded balances
- **Compliance**: Private Proofs of Innocence (PPOI)
