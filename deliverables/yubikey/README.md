# YubiKey (Yubico)

## Description
YubiKey is a hardware security key manufactured by Yubico that provides strong two-factor, multi-factor, and passwordless authentication. Founded in 2007 by Stina and Jakob Ehrensvärd in Sweden, Yubico co-authored the FIDO U2F standard with Google and has become the industry standard for phishing-resistant authentication. The company went public on Nasdaq Stockholm in September 2023.

## Key Features

### Authentication Protocols
- **FIDO2/WebAuthn**: Hardware-bound passkeys, discoverable credentials
- **FIDO U2F**: Universal 2nd Factor
- **PIV (Smart Card)**: NIST standard
- **OpenPGP**: RSA up to 4096 bits, ECC up to 384 bits
- **Yubico OTP**: One-time passwords
- **OATH-TOTP/HOTP**: Time and event-based codes

### Hardware
- IP68 rated (water/dust resistant)
- Crush resistant, no batteries, no moving parts
- Models: YubiKey 5 Series, YubiKey Bio (fingerprint), Security Key Series
- Entry price: ~$25-75 depending on model
